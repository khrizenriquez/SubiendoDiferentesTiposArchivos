SubiendoDiferentesTiposArchivos
===============================

Subiendo archivos de tipo txt, pdf y excel
//---------------------------------------------------

El objetivo es poder subir distintos formatos de archivo al servirdor, en este ejemplo veremos como subir los archivos con extensiones pdf, txt, xls(doc de word es parecido solo debemos colcoar el nombre correcto en vez de que sea a Excel es a Word).

Tenemos una página que muestra la información que se encuetra en la base de datos, esta misma información la migraremos a un formato pdf, archivo plano o de hoja de cálculo presionando un simple botón que hára todo el trabajo por nosotros. Este ejemplo esta creado con CSS, HTML y PHP principalmente, se tomo como base un ejemplo de Cesar Cancino http://www.cesarcancino.com/.
